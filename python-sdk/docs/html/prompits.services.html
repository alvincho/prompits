class prompits.services.Service.Service(name: str, description: str = None)[source]

Bases: Pit, ABC

Service is a specialized abstract Pit that provides functionality to agents.

A Service is owned by the agent that created it and can be used by other agents depending on access permissions. Services can provide various capabilities like API access, job scheduling, or data processing.
