class prompits.pools.DatabasePool.DatabasePool(name: str, description=None, connectionString=None)[source]

Bases: Pool, ABC

Abstract base class for database-specific pool implementations.

DatabasePool provides a common interface for interacting with different database systems. It defines methods for executing queries, managing transactions, and performing CRUD operations.

This class should be subclassed to implement specific database adapters.

Custom JSON encoder that handles datetime objects and other non-serializable types.

This encoder extends the standard JSONEncoder to properly serialize: - datetime objects (converted to ISO format strings) - methods and functions (converted to string representations) - custom objects (converted to dictionaries of their public attributes)
