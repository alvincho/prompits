class prompits.Agent.Agent(name: str = 'Agent', description: str = None, agent_id: str = None)[source]

Bases: Pit

The main agent class in the Prompits framework.

An Agent is a specialized Pit that can instantiate multiple other Pits, communicate with other agents, and participate in plazas. It manages plugs for communication, pools for data storage, and plazas for agent discovery and collaboration.

The Agent class is the central component that brings together all the pieces of the Prompits framework.

UsePracticeRemote(practice: str, agent_address: str, practice_input: dict = None)[source]

Use a practice from a remote agent.

This method allows an agent to call a practice on another agent, either local or remote. It handles the communication details, including message formatting, sending, and response handling.

The method supports three modes of operation: 1. Local practice calls (when agent_id matches self.agent_id) 2. Testing mode with simulated responses (when self.testing_mode is True) 3. Production mode with actual remote calls (default when not in testing mode)

Parameters:

  • practice – The practice to use, can be in the format “pit_name/practice_name” or just “practice_name”

  • agent_address – The address of the agent to call in the format “agent_id@plaza_name”

  • practice_input – Dictionary containing input parameters for the practice

Returns:

A dictionary containing the result of the practice or error information

Success case: {“result”: <result_value>, …} Error case: {“error”: <error_message>}

Return type:

dict

Raises:

No exceptions are raised; errors are returned in the result dictionary –
